# cage-history

`cage-history` defines the conversation history conventions for CAGE-enabled projects. It establishes how AI agents record, reference, and reason about prior interactions within a project.

> **Status:** This package is under active development. Conventions are not yet defined.

## Installation

```bash
npm install @dleangen/cage-history
```

## CAGE ecosystem

| Package | Description |
|---------|-------------|
| [cage-core](https://github.com/dleangen/cage-core) | Agent protocol specification |
| [cage-cli](https://github.com/dleangen/cage-cli) | CLI for scaffolding CAGE-enabled projects |
| **cage-history** | Conversation history conventions (this package) |
| [cage-management](https://github.com/dleangen/cage-management) | Management tooling for CAGE projects |
